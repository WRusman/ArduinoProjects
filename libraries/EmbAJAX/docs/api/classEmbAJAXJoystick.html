Detailed Description

This class provides a basic joystick for directional control. WORK IN PROGRESS, API and behavior may be subject to change in future versions of EmbAJAX.

active_timeoutMinimum timeout between two position change notifications (in milliseconds)
idle_timeoutTimeout after which the position other than center will be regarded as uncertain (e.g. due to unreliable network connection). : While the position is off-center in the client, updates will be send at half this timeout.
position_adjustCustom javascript that will be applied to "correct" the user supplied position, e.g. snapping it to certain allowed positions. e.g. EmbAJAXJoystick_POSITION_9_DIRECTIONS
snap_backCustom javascript that will be applied to snap back the position on mouse release. EmbAJAXJoystick_SNAP_BACK
